## 💡 Code Style in This Repository

Every solution file follows a consistent, professional format:

```python
# =============================================================
# File      : 05_find_a_string.py
# Chapter   : 03 Strings
# Topic     : Finding a substring within a string
# Platform  : HackerRank — Python Domain
# =============================================================

# Problem Statement:
# ------------------
# In this challenge, the user enters a string and a substring.
# Your task is to find out how many times the substring
# appears in the given string. Count overlapping occurrences.

# Key Concepts:
# - str.find() : returns index of first occurrence, -1 if not found
# - Manual sliding window approach for overlapping matches

# ---------------------------------------------------------
# Solution
# ---------------------------------------------------------

def count_substring(string, sub_string):
    count = 0
    start = 0

    while start < len(string):
        # find() searches from index `start` onward
        pos = string.find(sub_string, start)

        if pos == -1:
            break          # no more occurrences found

        count += 1
        start = pos + 1    # move one step forward to allow overlapping matches

    return count


if __name__ == "__main__":
    string = input()
    sub_string = input()
    print(count_substring(string, sub_string))
```
